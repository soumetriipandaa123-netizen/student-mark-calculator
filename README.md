# Student Grade Calculator

## Overview

The Student Grade Calculator is a web-based project developed using HTML, CSS, and JavaScript. It provides an interactive interface for calculating and displaying student results based on marks entered for five subjects.

## Features

* Total Marks Calculation
* Percentage Calculation
* Grade Display
* Pass/Fail Status Display
* Input Validation
* User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
